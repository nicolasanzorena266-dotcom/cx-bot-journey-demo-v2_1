# Personal CX Lab · Journey Demo

Demo web estática para simular recorridos conversacionales y generar una card operativa para el asesor.

## Flujos incluidos

- Ventas
- Onboarding
- Soporte
- Prepago *(visible, pendiente de definición)*
- Overnight *(visible, pendiente de definición)*
- Retención

Personal Pay fue retirado de esta versión.

## Qué muestra

- Chat estilo WhatsApp con el recorrido del cliente.
- Selector interno de flujo a simular.
- Card del asesor con datos capturados durante el recorrido.
- Datos a consultar en CRM/herramientas cuando corresponde.
- Acción sugerida por flujo.
- Métricas del recorrido.
- Exportación CSV.

## Cómo subirlo a GitHub Pages

1. Descargar y descomprimir el ZIP.
2. Entrar al repositorio actual.
3. Subir estos archivos reemplazando los existentes:
   - `index.html`
   - `style.css`
   - `flow.js`
   - `context-pass.js`
   - `forensics.js`
   - `script.js`
   - `README.md`
4. Confirmar el commit.
5. Esperar 1 o 2 minutos y recargar el link de GitHub Pages.

## Nota

No usa datos reales ni se conecta a sistemas productivos. Es una demo conceptual para presentar journeys, captura de contexto y handoff al asesor.
