# Personal CX Lab Demo

Demo interactiva para simular recorridos de atención y visualizar una card operativa para el asesor.

## Flujos incluidos

- Ventas
- Onboarding
- Soporte
- Prepago (pendiente de definición)
- Overnight (pendiente de definición)
- Retención
- Personal Pay figura visualmente como fuera de alcance y no tiene flujo activo.

## Uso

Abrir `index.html` en navegador o publicar el repositorio con GitHub Pages.

## Archivos

```txt
index.html
style.css
flow.js
context-pass.js
forensics.js
script.js
README.md
```

## Nota

Demo conceptual. No usa datos reales ni se conecta a CRM, herramientas internas o sistemas productivos.
